## Introduction

Warp is a modern terminal built for productivity, and mastering its keyboard shortcuts can significantly enhance your workflow. This guide provides a comprehensive reference for all keyboard shortcuts available in Warp, organized by function.

## Navigation Shortcuts

### Cursor Movement

| Shortcut | Action |
|----------|--------|
| `Ctrl + A` | Move cursor to beginning of line |
| `Ctrl + E` | Move cursor to end of line |
| `Alt + B` | Move cursor one word backward |
| `Alt + F` | Move cursor one word forward |
| `Ctrl + B` | Move cursor one character backward |
| `Ctrl + F` | Move cursor one character forward |

### Command History

| Shortcut | Action |
|----------|--------|
| `Ctrl + R` | Search command history |
| `Ctrl + P` | Previous command in history |
| `Ctrl + N` | Next command in history |
| `Alt + .` | Insert last argument of previous command |

### Terminal Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl + L` | Clear terminal screen |
| `Cmd + K` | Clear terminal buffer |
| `Cmd + ↑` | Scroll up |
| `Cmd + ↓` | Scroll down |
| `Cmd + Page Up` | Scroll one page up |
| `Cmd + Page Down` | Scroll one page down |

## Text Editing Shortcuts

### Selection

| Shortcut | Action |
|----------|--------|
| `Shift + →` | Select one character right |
| `Shift + ←` | Select one character left |
| `Shift + Alt + →` | Select one word right |
| `Shift + Alt + ←` | Select one word left |
| `Cmd + A` | Select all text in current prompt |

### Copy/Paste

| Shortcut | Action |
|----------|--------|
| `Cmd + C` | Copy selected text |
| `Cmd + V` | Paste text at cursor position |
| `Cmd + Shift + C` | Copy command output |
| `Ctrl + W` | Cut word before cursor |
| `Ctrl + K` | Cut text from cursor to end of line |
| `Ctrl + Y` | Paste text cut by previous command |

### Deletion

| Shortcut | Action |
|----------|--------|
| `Ctrl + H` | Delete character before cursor |
| `Ctrl + D` | Delete character under cursor |
| `Alt + D` | Delete word after cursor |
| `Ctrl + W` | Delete word before cursor |
| `Ctrl + U` | Delete from cursor to beginning of line |
| `Ctrl + K` | Delete from cursor to end of line |

## Tab & Window Management

### Tab Controls

| Shortcut | Action |
|----------|--------|
| `Cmd + T` | Open new tab |
| `Cmd + W` | Close current tab |
| `Cmd + Shift + ]` | Next tab |
| `Cmd + Shift + [` | Previous tab |
| `Cmd + 1-9` | Switch to tab by number |

### Window Controls

| Shortcut | Action |
|----------|--------|
| `Cmd + N` | Open new window |
| `Cmd + Shift + N` | Open new window with clean environment |
| `Cmd + Q` | Quit Warp |
| `Cmd + M` | Minimize window |

### Split Panes

| Shortcut | Action |
|----------|--------|
| `Cmd + D` | Split pane vertically |
| `Cmd + Shift + D` | Split pane horizontally |
| `Cmd + Shift + Arrow` | Navigate between panes |
| `Cmd + W` | Close current pane |

## Command Blocks

### Block Selection

| Shortcut | Action |
|----------|--------|
| `Shift + ↑/↓` | Extend selection to previous/next block |
| `Alt + Click` | Select specific block |
| `Cmd + Shift + A` | Select all blocks |

### Block Manipulation

| Shortcut | Action |
|----------|--------|
| `Cmd + Enter` | Rerun selected block |
| `Cmd + Shift + Enter` | Edit and run selected block |
| `Cmd + Delete` | Clear selected block |
| `Cmd + /` | Toggle block comment |

## Warp AI Commands

### AI Completion

| Shortcut | Action |
|----------|--------|
| `Tab` | Accept AI suggestion |
| `Esc` | Dismiss AI suggestion |
| `→` | Accept word by word |
| `Alt + ]` | Cycle through AI suggestions |

### AI Assistance

| Shortcut | Action |
|----------|--------|
| `Cmd + Shift + L` | Open Warp AI command palette |
| `@` | Trigger AI commands menu |
| `Cmd + \` | Toggle AI assistant panel |
| `Cmd + Shift + R` | Request AI explanation of command |

## Customization

### Settings Access

| Shortcut | Action |
|----------|--------|

| `Cmd + J` | Toggle command palette |
| `Cmd + Shift + P` | Open actions menu |

### Theme Controls

| Shortcut | Action |
|----------|--------|

| `F11` | Toggle full screen |
| `Cmd + +` | Increase font size |
| `Cmd + -` | Decrease font size |
| `Cmd + 0` | Reset font size |

### Custom Shortcut Configuration

You can customize keyboard shortcuts in Warp by:
1. Opening Settings (`Cmd + ,`)
2. Navigating to Keyboard Shortcuts
3. Editing the existing shortcuts or adding new ones

For more detailed documentation, visit the [Warp Docs](https://docs.warp.dev/).
